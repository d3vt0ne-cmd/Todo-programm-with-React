TODO-приложение для управления задачами. Реализовано на React с использованием хуков и localStorage.

## Цели выполненной работы

**Цель:** Создать интерактивное TODO-приложение, демонстрирующее навыки работы с React: компоненты, состояние, события, локальное хранилище и UI-дизайн.

Выполнено:
- Базовая функциональность: добавление, удаление, отметка задач
- Отдельный компонент для задач
- Перемещение задач, чекбоксы, разделение на активные/выполненные
- Сохранение в localStorage
- Кнопка "Удалить все" с подтверждением
- Время создания задач
- Адаптивный дизайн

## Запуск
npm install  
npm run dev

Открыть: http://localhost:5173
